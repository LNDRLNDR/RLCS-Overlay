# RLCS Overlay & Game HUD

Showcase of custom broadcasting overlays and in-game HUDs built for RLCS-competing teams. Each overlay replaces the default Rocket League HUD with a team-branded design and adds real-time boost meters, a minimap during goal replays, and post-game statistics.

**[View the live page](https://lndrlndr.github.io/RLCS-Overlay/)**

## Teams featured

- Gen.G Mobil1 Racing: Formula 1 racing theme with speed meter, animated sponsor box, and replay minimap
- Oxygen Esports (OXG)
- Team BDS
- PSG Tundra: co-branded Paris Saint-Germain and Tundra design
- Team Vitality: two-time RLCS World Champions
- Quadrant: co-founded by Lando Norris and Veloce Esports
- Akrew
- Resolve
- G1 / Gamers First: NFL-inspired overlay built for the G1 Invitational
